Geolocation API

https://developer.mozilla.org/ja/docs/Web/API/Geolocation_API

timestamp について

timestamp は DOMTimestamp を Date クラスに入れて、 toLocaleDateString(), toLocaleTimeString()を連結してます。

テーブルヘッダ説明

テーブルヘッダは下記のような意味を持ちます。

timestamp
タイムスタンプ
latitude[deg]
緯度[度]
longitude[deg]
経度[度]
accuracy[m]
緯度経度の精度[メートル]
altitude[m]
海面からの相対高度[メートル]
altitudeAccura
海面からの相対高度の精度[メートル]
heading[deg]
真北を基準とした端末の向き[度]
speed[m/s]
端末の移動速度[メートル毎秒]
